nodejs深入系列--util

发表于 2018-02-19   |   分类于 Javascript

util模块是一个工具模块。

util.inspect()

返回一个对象反序列化形成的字符串。还可在调试时带上颜色
即将对象转化为字符串

让控制台输出时带上各种数据类型各自的颜色：

util.format()

返回一个使用占位符格式化的字符串。可使用的占位符有%s、%d、%j
即让参数以某种方式输出。

util.log()

在控制台输出，类似于console.log()，但这个方法带有时间戳
