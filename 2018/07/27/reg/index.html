常见正则表达式

发表于 2018-07-27   |   分类于 Javascript

正则基础

字符匹配

  1. […]
    查找方括号之间的嗯任何字符
  2. [^…]
    查找任何不在方括号之间的字符
  3. [a-z]
    查找任何从小写a到小写z的字符
  4. [A-Z]
    拆招任何从大写A到大写Z的字符
  5. .
    查找单个字符，除了换行和行结束符
  6. \w
    查找单词字符，等价于[a-zA-Z0-9]
  7. \W
    查找非单词字符，等价于[^a-zA-Z0-9]
  8. \s
    查找空白字符
  9. \S
    查找非空白字符
  10. \d
    查找数字，等价于[0-9]
  11. \D
    查找非数字，等价于[^0-9]
  12. \b
    匹配单词边界
  13. \r
    查找回车符
  14. \t
    查找制表符
  15. \0
    查找NULL字符
  16. \n
    查找换行符

重复字符匹配

  1. {n, m}
    匹配前一项至少n次，但不能超过m次
  2. {n,}
    匹配前一项至少n次
  3. {n}
    匹配前一项n次
  4. n?
    匹配前一项0次或1次
  5. n+
    匹配前一项1次或多次
  6. n*
    匹配前一项任意次
  7. n$
    匹配任意结尾为n的字符串
  8. ^n
    匹配任意开头为n的字符串
  9. ?=n
    匹配任何其后紧接指定字符串n的字符串
  10. ?!n
    匹配任何其后没有紧接指定字符串n的字符串

匹配特定数字

  1. ^[1-9]\d*$
    匹配正整数
  2. ^-[1-9]\d*$
    匹配负整数
  3. ^-?[0-9]\d*$
    匹配整数
  4. ^[1-9]\d|0$
    匹配非负整数
  5. ^-[1-9]\d*|0$
    匹配非正整数
  6. ^[1-9]\d.\d|0.\d[1-9]\d$
    匹配正浮点数
  7. ^-([1-9]\d.\d|0.\d[1-9]\d)$
    匹配负浮点数
  8. ^-?([1-9]\d.\d|0.\d[1-9]\d)$
    匹配浮点数
  9. ^[1-9]\d.\d|0.\d[1-9]\d|0?.0+|0$
    匹配非负浮点数
  10. ^(-([1-9]\d.\d|0.\d[1-9]\d))|0?.0+|0$
    匹配非正浮点数

匹配特定字符串

  1. ^[A-Za-z]+$
    匹配26个英文字母组成字符串
  2. ^[A-Z]+$
    匹配26个大写英文字母组成字符串
  3. ^[a-z]+$
    匹配26个小写英文字母组成字符串
  4. ^[A-Za-z0-9]+$
    匹配由数字和26个英文字母组成的字符串
  5. ^\w+$
    匹配有数字、26个英文字母或者下划线组成的字符串

正则方法

test

检索字符串中指定值。返回true或false

exec

检索字符串中指定值。返回一个数组，其中存放匹配的结果。若未找到，则返回null

数组的属性

数组可以添加属性，而且会出现在数组中，但是不能直接通过数组位数去调用，可直接通过属性去调用。

search

检索字符串中指定值。返回匹配的字符串的起始位置，若不匹配则返回-1。
search不执行全局匹配，会忽略g

match

检索字符串中指定值。返回一个数组，其中存放匹配的结果。
和exec不同的是，g情况下match会返回所有匹配的结果，而exec只返回一个。

replace

替换匹配正则的字符串。

replace中的特殊字符

  1. $1, $2, …, $99
    分别代表字符串中与正则子表达式相匹配的项。

  2. $&
    匹配的正则子表达式然后通过&添加其他字符串

  3. $`
    匹配正则子表达式左侧内容

  4. $’
    匹配正则子表达式右侧内容

  5. $$
    直接替换为$

replace函数参数

replace的第二个参数，也可以为函数。
函数的参数为固定形式：reg, 子表达式匹配的字符，匹配的位置，字符串本身

split

通过正则匹配进行分割字符串。
