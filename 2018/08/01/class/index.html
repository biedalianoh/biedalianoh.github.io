class分析与extends、super应用

发表于 2018-08-01   |   分类于 Javascript

class

类的示例：

static

类中static修饰的方法为静态方法。
使用static之后，注意事项：

  1. 静态方法只能被类本身调用，或者被继承之后的子类本身调用。
  2. 类中其他非静态方法调用静态方法时，不能使用this来调用，只能使用类名或者constructor来调用。
  3. 类中其他静态方法调用静态方法时，可直接使用this来调用

super

super有两种用法：

作为函数形式调用–super()

代表父类的构造函数。相当于把父类的所有属性和方法，都传递给子类。
继承时，必须在子类的构造函数中使用super()，否则新建实例时会报错。因为子类没有自己的this对象，而是继承自父类的this对象，然后对其进行加工。若不调用super()，子类就得不到this对象。

作为对象形式调用–super

super不能单独出现在语句中。如不能用console.log(super)，只能使用super.methodA()这种形式。

  1. 普通方法中
    指向父类的原型对象。所以contructor中定义的属性super是访问不到的。

  2. 静态方法中
    指向父类。指向父类后，意味着super只能调用父类的静态方法。父类的属性和原型方法无法调用。

new操作符

new操作符，将创建一个新对象，并把新对象的__proto__链接到构造函数的prototype。并且this将通过call形式被绑定到新对象上。
