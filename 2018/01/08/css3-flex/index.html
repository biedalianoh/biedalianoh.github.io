flex布局

发表于 2018-01-08   |   分类于 CSS

传统布局–盒状模型，依赖display + position + float属性。
弹性布局–flex，能更简便、完整响应各种页面布局

兼容性

IE10部分支持，需要-ms-前缀
chrome部分支持，需要-webkit-前缀
safari部分支持，需要-webkit-前缀

外部容器

方向

flex-direction

设置内部内容的排列方向

  1. row(默认值)
    水平方向，起点为左
  2. row-reverse
    水平方向，起点为右
  3. column
    垂直方向，起点为上
  4. column-reverse
    垂直方向，起点为下

换行

flex-wrap

设置内部内容一行排不下，如何换行

  1. nowrap(默认值)
    不换行
  2. wrap
    换行。往下换行
  3. wrap-reverse
    换行。往上换行

对齐

justify-content

设置内容横轴对齐方式

  1. flex-start(默认值)
    左对齐
  2. flex-end
    右对齐
  3. center
    居中
  4. space-between
    两端对齐
  5. space-around
    均匀分布

align-items

设置内容纵轴对齐方式

  1. flex-start
    上对齐
  2. flex-end
    下对齐
  3. center
    垂直居中
  4. baseline
    第一行文字下基线对齐
  5. stretch(默认值)
    占满整个容器的高度

外部容器简写

flex-flow

是flex-direction和flex-wrap属性的简写。即设置排列方向，和设置换行
默认值为row nowrap

align-content

是justify-content和align-items的简写。即设置横轴对齐方式，和纵轴对齐方式

内部项

排列

order

定义排列顺序，数值越小，越靠前。默认为0

放大缩小

flex-grow

设置放大比例。默认为0，即不放大

flex-shrink

设置缩小比例。默认为1，即若空间不足，将缩小

flex-basis

设置单项占据空间。默认值为auto，即自适应的大小。
若设置具体值，则项目将占据固定空间。如350px

对齐

align-self

设置单个项的对齐方式。
外部容器设置了对齐方式后，可用此设置单个项的对齐方式。

auto表示继承父元素的align-item属性，若无父元素，则等同于stretch

内部项简写

flex

是flex-grow flex-shrink flex-basis的简写。默认值为0 1 auto。
即放大、缩小、单项占据空间缩写。
后两个属性可选。

该属性有两个快捷值：
auto—-表示 1 1 auto
none—-表示 0 0 auto
