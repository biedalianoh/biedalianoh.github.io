for, forEach, $.each, _.forEach中的跳出循环

发表于 2018-01-30   |   分类于 Javascript

for

for支持全部的跳出方法：
continue – 跳出当前回合到下一回合
break – 跳出整个循环
return – 跳出整个循环，并且返回值

forEach

只支持return
return – 跳出当前回合，并且return不会有任何返回值

$.each

只支持return
return false – 跳出整个循环，相当于break
return true – 跳出当前回合，相当于continue

_.forEach

只支持return
return false – 跳出整个循环，相当于break
return true – 跳出当前回合，相当于continue
