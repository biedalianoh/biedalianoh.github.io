Javascript基础--函数和定义优先级

发表于 2017-01-11   |   分类于 Javascript

Javascript中的包括变量的定义，函数的定义；函数的静态方法，原型方法；变量的前置定义，后置定义；运算符的优先级；变量的提升，等等经常容易发生混淆。避免在前端开发过程中再踩这些类似的坑，在此总结。

js基本数据类型

js基本数据类型有六种：Undefined, Null, 布尔值， 字符串， 数值， 对象
判断的基本方式是，他们属于原始的基本类型，而不是对象，不能使用 new 命令。
ES6 新增了一种基本数据类型： Symbol

静态方法和原型方法

静态方法的优先级高于原型方法。

所有的声明变量或声明函数都会被提升到当前函数的顶部。

自执行函数的this

若构造函数的执行，不是通过new方法定义的对象调用，则this为window

构造函数的返回

在传统语言中，构造函数不应该有返回值。
若有返回值，则分以下两种情况。

返回基本类型

如返回 string, number, boolean, null, undefined, 则实际效果和 无返回值 一样。返回的是实例化对象，包括构造函数的属性和prototype

另外，关于构造函数的属性和原型：
当实例化对象时，并不能实例化非this添加的属性和方法。
原型prototype可以实例化。
