CSS3中的Matrix矩阵

发表于 2017-01-11   |   分类于 Css

transform中的Matrix

CSS3中的矩阵是指 matrix() 和 matrix3d()，前者是元素2D平面的移动变换，后者是3D变换。2D变换为 33， 3D变换则是 44 矩阵。

matrix实际上能设置的属性有：skew(拉伸)，scale(缩放)，rotate(旋转)，translate(位移)。
即matrix是以下属性的集合：

transform-origin

在理解matrix之前，先了解一个属性：transform-origin。
即矩阵所有相关变化的基准点。默认值为 0
在单独使用skew，scale，rotate，translate时，若设置此值则变化的基准点响应改变。transform-origin对matrix无效。

matrix 缩放

transform: matrix(1, 0, 0, 1, 8, 9) 中第一个和第四个参数对应x轴，y轴缩放比例。
单独使用 transform: scale(a, b)时，需要乘以transform-origin(x% y%)中对应的x，y值。

matrix 旋转

matrix(cosθ,sinθ,-sinθ,cosθ,0,0)
由于matrx的旋转比较麻烦，而且 transform: rotate(deg) 格式要比matrix简单多了，所以不推荐用matrix写旋转。

matrix 拉伸

matrix(1,tan(θy),tan(θx),1,0,0)
即通过设置第二个和第三个值，来设置skew拉伸。

综上所述

综上所述，matrix的6个值可以同时设置 位移(translate), 缩放(scale), 拉伸(scale) 属性，旋转需要单独设置。因此，在不变更translate-origin时，若需要使用 位移(translate), 缩放(scale)功能时，拉伸(skew)可以使用matrix进行简写，其他情况则使用translate单独的方法。
