组件

前面已经介绍了组件的几个常用方法：createClass, render, getInitialState.
现在介绍组件生命周期（挂载，更新，卸载）常用的其它方法。
挂载：

getInitialState() 是第一个被调用的，会在React将组件插入DOM之前被调用。
componentWillMount() 是第二个被调用的，会在React将组件即将插入DOM时被调用。
render() 是第三个被调用的，将组件插入DOM元素。
componentDidMount() 是第四个被调用的，插入完成。

更新：

componentWillReceiveProps() 更新阶段第一个被调用。组件从父组件接收到信属性时被调用
shouldComponentUpdate() 更新阶段第二个被调用。 决定组件是否重新渲染
componentWillUpdate() 更新阶段第三个被调用。即将更新DOM之前被调用。
render() 更新阶段第四个被调用。更新DOM。
componentDidUpdate() 更新阶段第五个被调用。更新DOM完成之后立即执行。

卸载：

componentWillUnmount() 组件被删除之前触发的方法。

componentDidMount

componentDidMount() 仅仅会在初始化渲染完成之后调用一次。此时React已经创建了DOM树。
它最适合用来整合React和其它js库。

componentWillMount

componentWillMount() 会在React将组件即将插入DOM时被调用。

componentWillUnmount

componentWillUnmount() 方法在组件即将被卸载之前会被调用。

Action对象也具有新 state 的属性。

存储的作用是 储存，管理数据，同时对外向React组件提供数据。
