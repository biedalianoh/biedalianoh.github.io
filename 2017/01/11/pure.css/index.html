Pure.css栅格化Grids应用提取

发表于 2017-01-11   |   分类于 CSS

About

雅虎响应式css前端框架，建立在Normalize.css基础上。由于每家公司在定制网站UI时，会有多种框架或自定义样式组合，全部应用特定css框架的情况比较少见，栅格化系统是现代网站中比较常用的功能，pure.css提供了5格和24格网格功能，可单独提取其中的Grids作为常备插件使用。

Grids

网格系统，Pure网格系统支持5格和24格。基本满足大部分使用场景，雅虎还提供了一个定制化网格工具Pure Grids Rework Plugin,感兴趣的同学可以自己研究一下。

How To Use

我抽取了 pure.css 中的 .pure-g 和 .pure-u 相关的css属性，并组合 grids-responsive.css，合并压缩成了pure-grids-min.css，可下载使用此css组件。

1. pure-g

网格样式，在网格系统外面添加class=”pure-g”定义一个网格，表明此处需要使用网格系统。

2. pure-u-*

单元格样式。如pure-u-1-2表示单元格占1/2的宽度，pure-u-2-3表示单元格占2/3的宽度。

3. sm, md, lg, xl

此处 1em == 16px

  • sm： 当页面宽度 >= 568px 时触发 @media screen and (min-width: 35.5em) .pure-u-sm-*
  • md： 当页面宽度 >= 768px 时触发 @media screen and (min-width: 48em) .pure-u-md-*
  • lg： 当页面宽度 >= 1024px 时触发 @media screen and (min-width: 64em) .pure-u-lg-*
  • xl： 当页面宽度 >= 1280px 时触发 @media screen and (min-width: 80em) .pure-u-xl-*
