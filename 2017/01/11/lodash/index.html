Lodash实用中文api

发表于 2017-01-11   |   分类于 Javascript

Lodash.js是一个现代模块化性能良好的Javascript工具库。和Underscore功能类似，并提供了更为丰富的API。
近期会持续更新一份实用的lodash使用参考（v4.6），数组部分参考了cnblogs的shikelong（v3.8），这拼音和我一个同事kelong的还挺像的，难道是他？

lodash下载

可通过官网或cdn下载。也可通过npm或bower等加载。

数组基本操作

查找数组元素

first–返回第一个元素

head–返回第一个元素

tail–返回除第一个元素之外的数组

last-返回最后元素

findIndex–从头查询元素序号

遍历，查询到符合的元素即返回序号，没有符合的返回-1。predicate为Function|Object|String。

findLastIndex–从尾查询元素序号

用法类似 _.findIndex()，不过返回的序号为符合要求的最后一个。

indexOf–从头查询元素序号

从数组中查询value的序号，参数3如果是number，则从数组的number位置开始查找，参数3如果是true，则执行二分查找（二分查找为升序或降序的有序列表，查找时从中匹配为=或>或<，则可直接查找成功或只查找折半的子表）。

lastIndexOf–从尾查询元素序号

功能和_.indexOf()类似，不过从末尾开始查询。

替换数组元素

fill–替换元素

从start到end，用value修改或填补数组元素。此方法直接改变数组，而不是返回新的数组。

删除数组元素

initial–删除末尾元素

reset–删除首元素

和_.initial()相反。

drop–从头删除元素

从头开始删除number个数组元素。number不传默认为1

dropRight–从尾删除元素

用法和drop一样，不过从尾部开始删除元素

dropRightWhile–从尾多功能过滤元素

从尾部开始通过参数对数组进行过滤，返回符合条件的调用。predicate可以为 Function|Object|String 。

dropWhile–从头多功能过滤元素

功能和_.dropRightWhile()一样，只不过从头开始过滤。

compact–去除假值

去除数组中boolean值为false的元素，返回新的数组

difference–删除特定元素

在数组array中去除和数组values中元素相同的元素。

differenceBy–多功能删除特定元素

通过参数Function|Object|String来进行参数匹配，删除相应元素。

differenceWith–多功能删除特定元素

通过指定比较方式删除所匹配的数组元素。

pull–删除特定元素

直接对原数组操作，不返回新数组。

pullAll–删除特定元素

和pull功能一样，不过pullAll参数为数组。

pullAllBy–多功能删除特定元素

匹配参数Function|Object|String删除对应的元素。

pullAllWith–多功能删除特定元素

删除匹配指定匹配方式的元素。

pullAt–删除特定序号元素

直接对原数组操作，并且返回移除的元素组成的数组。

remove–多功能删除元素

逻辑和_.dropRightWhile()类似，通过Function|Object|String进行删除元素匹配。对原数组进行操作，并返回移除的元素组成的数组。

``bash
var array = [1, 2, 3, 4];
var delArr = _.remove(array, function(item){
return item % 2 == 0;
});
//array: [1, 3]
//delArr: [2, 4]

xor–删除重复值后合并

可多组数组进行，两两配对删除，然后合并。

xorBy–多功能删除重复值后合并

xorWith–多功能删除重复值后合并

uniq–数组去重

如isSorted设置为true，表明数组为升降序排列数组，使用二分法去重。最后一个参数thisArg为指定this指向。

uniqBy–数组多功能去重

可添加数据处理后对比去重。第二个参数可为 Function|Object|String。

uniqWith–数组多功能去重

var objects = [{ ‘x’: 1, ‘y’: 2 }, { ‘x’: 2, ‘y’: 1 }, { ‘x’: 1, ‘y’: 2 }];
.uniqWith(objects, .isEqual);
// [{ ‘x’: 1, ‘y’: 2 }, { ‘x’: 2, ‘y’: 1 }]

sortedUniqBy–有序数组多功能去重

和_.uniqBy()功能类似，不过只对有序数组去重。

添加数组元素

sortedIndex–对有序数组进行插入

把元素按有序排列的方式在相应位置插入一个有序数组，返回插入的位置。

sortedLastIndex–对有序数组进行反向插入

与_.sortedIndex()功能类似，不过是从尾部开始插入。

拆分数组

chunk–对数组分块

根据number对array进行均等分块

合并数组

concat–合并数组或添加元素

添加任何元素至数组中，不改变原数组，返回一个新数组。

flatten–抹平嵌套数组

isDeep为空或者false时，只抹平第一层嵌套。为true时，全部抹平。

flattenDeep–全部抹平嵌套数组

flattenDepth–抹平嵌套数组n次

默认depth为1.

join–合并数组为字符串

默认合并字符为 , 。

数组排序

reverse–反转数组

对原数组进行操作，不返回新数组。

获取子数组

slice–截取数组

和原生slice功能类似。

take–正向获取n个元素

获取数组中的前n个元素。默认为1

takeRight–反向获取n个元素

获取数组中的后n个元素。默认为1

takeRightWhile–反向多功能获取n个元素

通过Function|Object|String来确定所获取的元素。

takeWhile–正向多重获取n个元素

类似_.takeRightWhile()，执行顺序相反。

函数和方法

bind–指定函数的this

创建一个新的函数，使func的this指向为thisArg。[partials]为func所需的参数。

bindAll–绑定对象方法的this至对象本身

绑定object对象中的方法的this为object。若没有指定方法，则绑定object对象所有的方法的this至object。

bindKey–赋予Json中的方法到新方法

创建一个新的函数，把Object中的对应key的方法赋予新的函数，并且可以传参。和bind不同，bindKey不改变函数的this指向。

before–限定函数执行次数

func只能执行n-1次，第n次开始不会执行。

after–限定函数从第n次执行

和_.before()相反，func调用第n次的时候，才会开始执行。

ary–限定函数参数个数

创建一个函数，执行限定参数个数为n的func。n默认为func参数个数。

unary–限定参数个数为1个

创建一个函数，最多接受一个参数，其他参数忽略。

curry–部分参数待定执行

创建一个新的函数，新的函数运行部分参数，再创建另一个函数，运行另外的参数，直到所有的参数凑齐，开始执行函数。

curryRight–部分参数反向待定执行

和 _.curry() 类似，不同的是传入函数的参数顺序相反。

debounce–延迟防反复调用

当触发事件时，func会在给定的延迟时间wait之后进行调用，若在这个时间内又触发了事件，则延迟时间重置，直到延迟时间内没有重复调用时，执行func。适用于类似window.onresize, onscroll, onmousemove之类频繁触发的方法。

throttle–延迟防反复调用

功能和debounce类似。网上有个例子对比很准确：

defer–当前调用栈被清空执行

推迟执行 func 直到当前调用栈被清空。此外，在执行时额外的参数将会传入 func 。

delay–延迟执行函数

延迟wait毫秒数后执行func。

flip–参数倒序执行函数

创建一个函数，参数顺序倒转。

negate–创建执行结果相反函数

创建一个执行结果与函数 func 相反的函数。func 函数在执行时会绑定 this 并传入创建函数的参数。

once–只执行一次函数

创建一个只执行一次 func 的函数。重复调用函数将返回首次调用的值。func 在执行时将绑定 this(使用func本身的this), 并传入创建函数的参数。

overArgs–处理函数参数

通过方法处理对应参数之后，再进行参数调用。

partial–设定函数参数

和_.bind()类似，不过不改变func的this指向。

partialRight–设定函数反向参数

和_.partial()类似，不过反转参数顺序。

rearg–指定参数顺序

创建一个执行 func 时能够指定传入参数的位置的函数。

wrap–传递第一个参数给函数

把第一个参数传给后面的函数，函数也可添加自己的其他参数。

return _.filter(string, function(v) {
return /[aeiou]/i.test(v);
});
}

// → [‘e’]

_(‘fred’).vowels().value();
// → [‘e’]

.mixin({ ‘vowels’: vowels }, { ‘chain’: false }); (‘fred’).vowels();
