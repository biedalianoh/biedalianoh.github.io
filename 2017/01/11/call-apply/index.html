call, apply, callee, caller, bind应用

发表于 2017-01-11   |   分类于 Javascript

call

将函数绑定到另外一个对象上运行。
即函数通过绑定的对象为主体执行了一遍。

即将一个函数的上下文从初始的上下文改变为指定的thisArg。

不带参数arg

extend 中，将 this 改变为 base，将 c 改变为调用 base
即在 extend 中：
调用 this 时，都是调用 base
调用 c 时，都是调用 base。

带参数arg

print代替执行，并使用参数a, b, c, d

apply

和 call 用法一样，除了参数arg传递的方式。

caller

返回一个函数，这个函数是所指定的函数的调用者。若没有调用，则返回null。

即哪个函数调用的这个函数，即返回那个函数。

callee

返回函数自身。是arguments的一个属性。

可通过arguments.callee.length获取形参

bind

bind也是改变this的指向，但是不像call和apply一样立即执行一遍。bind只是改变。
