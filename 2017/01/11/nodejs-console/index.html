nodejs深入系列--console

发表于 2017-01-11   |   分类于 Javascript

nodejs中有两种console：

  1. console类。使用类似console.log(), console.error()之类的方法。
  2. 全局的console实例，用来写 stdout 和 stderr

同步和异步

console普通情况下是同步的。只有当console的目标是文件的时候是异步。

Console

创建一个console实例有两种方式：

  1. const Console = require(‘console’).Console;
  2. const Console = console.Console;

console.error([data][, …args])

console.info([data][, …args])

console.log([data][, …args])
