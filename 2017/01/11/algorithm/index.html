算法的时间复杂度

时间复杂度是总运算次数表达式中受n的变化影响最大的那一项（不含系数）。
如一个算法循环了 (n + n-1 + n-2 + n-3 +…+ 1) 等于 n * (n + 1) / 2，约等于 (n ^ 2) / 2，因为时间复杂度是不考虑系数的，所以时间复杂度是 O(n^2)
常见的时间复杂度（按复杂度递增排列）:
常数阶O(1), 对数阶O(log2n), 线性阶O(n), 限行对数阶O(nlog2n), 平方阶O(n^2), 立方阶O(n^3),指数阶O(2^n)

去重

去重算法。

  1. 设置两个循环，i和j，让初始j=i+1
  2. 然后让i的元素去和j的元素做对比，如果arr[i]=arr[j]，则让i加1，是为了让i的当前元素被过滤掉。
  3. 把当前的arr[i]添加至新数组中。
  4. 循环完成返回新的数组

二分法(折半查找)

查找对应值的index。必须为升序或降序排列。

  1. 设定开始的startIndex为0， 结束的endIndex为length-1，中间值middleIndex为floor((startIndex-endIndex)/2)
  2. 通过给定middleIndex对应的值是否相等，并且startIndex要小于endIndex来设置while循环。
  3. 在while循环中，不停使用二分来重置startIndex，endIndex，middleIndex，知道while条件不符，while循环结束
  4. 返回对应的arr[middleIndex]
