Angular 使用笔记

发表于 2017-01-11   |   分类于 Javascript

Angular当然是现在流行的前端框架之一，和React，Ember一起并称前端开发的三神器。不过由于前公司因为业务需要，使一直使用的是另外一个轻量级的MVVM框架Rivets，使得对angular一直闻其声，未尽用。趁此机会便想把angular学习应用了一遍，并对比工作中使用的rivets，熟悉MV*的框架结构。

安装部署

可通过官网或中文社区下载安装。

使用方法

angular提供了多种api方法，全部api可查看 api手册目录

构造应用

通过定义作用域，模块，和控制器来建立一个angular应用。

其中angular模块module和控制器controller可以位于不同的js文件中。

两种初始化方式

angular在DOMContentLoaded事件触发时执行。

  • 自动初始化
    通过ng-app找到根作用域。

或直接指定注入器。

  • 手动初始化
    当需要主动控制初始化过程，需要在angular编译页面之前做一些操作，则可以手动初始化。

表达式

angular表达式 使html上居有简单运算能力。
可支持数字的 + - * /，和 字符串相加

ng 指令

angular的指令是以 ng 作为前缀的 HTML 属性。也可使用 data-ng- 来让angular指令对html5有效(html5允许以 data- 扩展自定义属性).

ng-app

定义一个angular应用的根元素。

ng-init

初始化应用程序的一些变量。相当于把这个变量添加到 $scope 的属性上。

ng-bind

添加数字或字符串到html中。相比表达式添加，ng-bind在页面加载时不会出现源代码。

ng-model

把元素值绑定到$scope。用于表单项(input select textarea)。

angular为表单项的ng-model提供了四个状态值：
如表单：

  1. $valid
    输入合法，为true，否则为false
  2. $invalid
    和valid相反
  3. $dirty
    输入值改变了，为true，没有做过更改则为false
  4. $touched
    通过触屏点击了，则为true，否则为false
  5. $error
    判断是否输入正确。
    $error.required： 当没有填写时为true
    $error.email: 当邮箱格式不正确时为true

另外，angular为ng-model的状态，添加了对应的css类。

ng-model-options

设置ng-model绑定触发时的选项。

  1. updateOn
    触发事件更新绑定数据。

  2. debounce
    定义更新绑定数据的延迟时间。

  3. allowInvalid
    设置验证。

  4. timezone
    设置时间格式。

ng-mousedown

鼠标按下时触发。不会覆盖原生onmousedown事件。

ng-mouseenter

鼠标移过元素时触发。不会覆盖原生onmouseenter事件。

ng-mouseleave

鼠标离开元素时触发。不会覆盖原生onmouseleave事件。

ng-mousemove

鼠标在元素上移动时触发。不会覆盖原生onmousemove事件。

ng-mouseover

鼠标移动到元素上时触发。不会覆盖原生onmouseover事件。

ng-non-bindable

规定不编译当前元素和其子元素。

ng-open

设置detail列表的open属性。

ng-readonly

设置input或textarea是否只读。

ng-selected

设置select列表中的option的selected属性。即设置select中哪个option被选中。

ng-srcset

设置不同分辨率设备使用相应大小图片。
srcset为html5中img的新属性。

ng-srcset用法：

ng-style

设置元素行间样式。

ng-submit

提交表单。

ng-switch

根据表达式显示或隐藏对应的部分。
使用 ng-switch-when 和 ng-switch-default 进行控制。

ng-value

设置input或select元素的value值。

ng-repeat

迭代数组。

ng-options

创建一个下拉列表。数据源可为 数组，或 对象。

当然也可使用ng-repeat，但是当选择的值为对象时，即ng-model对应的selectedName为对象时，要用ng-options，因为ng-repeat的ng-model值为字符串。

ng-disabled

设置控件是否可用。

ng-show

设置标签隐藏或显示。

ng-hide

ng-click

设置点击事件。

ng-include

添加html文件内容。

ng-bind-html

添加html代码到标签中。

ng-bind-template

绑定多个表达式到标签中。

ng-blur

输入框失去焦点触发。

ng-change

输入框的值改变触发。

ng-checked

是否被选中。

ng-class

给元素绑定一个或多个类名。可添加字符串，对象，数组。

  1. 字符串。
    多个类名使用空格分隔。

  2. 对象。
    {key: value}。要添加的类名为 key, value为布尔值。只有当value为true时，添加key作为类名。

  3. 数组
    数组的每一个值都会设置为类名。

  4. 数组和对象混搭
    对象添加key为类名，数组元素添加本身类名。测试用例有问题，待解决。

ng-copy

html元素文本被拷贝时触发。不会覆盖原始的oncopy事件。二者都会执行

ng-cut

在剪切html元素文本时触发。不会覆盖原始的oncut事件。二者都会执行

ng-dblclick

双击时触发。不会覆盖原始的ondblclick事件。二者都会执行。

ng-focus

元素在获取焦点时触发。不会覆盖原始的onfocus事件。

ng-href

设置链接。

ng-if

是否移除html元素。

ng-keydown

按下时触发。不会覆盖onkeydown

ng-keypress。

点击时触发。不会覆盖onkeypress

ng-keyup

点击松开时触发。不会覆盖onkeyup

ng-list

字符串转为数组。通过逗号分隔数组元素。

ng-csp

设置网站内容安全策略。 来限制网站中是否可以包含某来源的资源。资源类型细分为 Script、Style、Image、Font、Connect、Media、Object、Frame 等
具体可见：浏览器安全策略 CSP 部署：teambition 实战

自定义指令

使用 .directive 函数添加自定义指令。
js中用驼峰法来命名自定义指令，如： myNg，但在html应用时，需要用 - 分割，如.
自定义指令有四种调用方法（调用方法见 方法-directive）:

  1. 元素名
  2. 属性
  3. 类名
  4. 注释

均会添加template中的内容。

angular方法

定义在全局函数angular上的方法。

angular.module()

定义一个angular的模块。
第二个参数的 [] 为定义模块的依赖。

angular.lowercase()

转换字符串为小写

angular.uppercase()

转换字符串为大写

angular.copy()

数组或对象深度拷贝

angular.forEach()

对象或数组的迭代函数

angular.isString()

判断给定的对象是否为字符串，如果是返回 true。

angular.isNumber()

判断给定的对象是否为数字，如果是返回 true。

angular.isArray()

如果引用的是数组返回 true

angular.isDate()

如果引用的是日期返回 true

angular.isDefined()

如果引用的已定义返回 true

angular.isElement()

如果引用的是 DOM 元素返回 true

angular.isFunction()

如果引用的是函数返回 true

angular.isObject()

如果引用的是对象返回 true

angular.isUndefined()

如果引用的未定义返回 true

angular.equals()

如果两个对象相等返回 true

angular.fromJSON()

反序列化 JSON 字符串

angular.toJSON()

序列化 JSON 字符串

angular.bootstrap()

手动启动 AngularJS

angular.element()

包裹着一部分DOM element或者是HTML字符串，把它作为一个jQuery元素来处理。

命名空间对象 $

命名空间对象须在创建controller时在参数中声明。

$scope

scope(作用域) 是应用在 HTML (视图) 和 JavaScript (控制器)之间的纽带。
在html中，无需添加 $scope 前缀，即可直接使用 $scope 的属性。

$rootScope

根作用域。每个ng-app有一个根作用域 $rootScope。 这个ng-app下创建的每个controller自己的 $scope 都继承自 根作用域 $rootScope。
可将 $rootScope 作为参数传递，在应用中使用。

$location

返回当前页面的 URL 地址。

$http

$http 是 AngularJS 应用中最常用的服务。$http向服务器发送请求，并响应服务器传送过来的数据。

其中 $http.get(url) 是用于读取服务器数据的函数。也可直接使用 .success() 做响应。

$timeout

延时处理。

$interval

计时器。

$index

$index 为 ng-repeat 中的序号。

$even 和 $odd

$even 为 ng-repeat 中判断为偶数。$odd 为奇数。

过滤器

AngularJS 过滤器用于转换数据。

currency

格式化数字为货币格式。

filter

从数组项中选择一个子集。

orderBy

根据某个表达式排列数组。

uppercase

格式化字符串为大写。

lowercase

格式化字符串为小写。
