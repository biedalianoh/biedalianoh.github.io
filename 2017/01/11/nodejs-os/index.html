发表于 2017-01-11   |   分类于 Javascript

os方法

os模块提供了一些操作系统相关的实用方法。

os.EOL

定义操作系统换行符。

os.arch()

操作系统的cpu架构。
可能的值为：’arm’, ‘arm64’, ‘ia32’, ‘mips’, ‘mipsel’, ‘ppc’, ‘ppc64’, ‘s390’, ‘s390x’, ‘x32’, ‘x64’, and ‘x86’
等同于 process.arch

os.constants

返回包含错误代码、进程信号等常用操作系统特定常量的对象

os.cpus()

返回每个cpu或者核心的安装信息。如：
