nodejs深入系列--stream

发表于 2017-01-11   |   分类于 Javascript

stream是nodejs中处理流数据的抽象接口。
nodejs中有多种stream对象。
所有的stream对象都是EventEmitter的实例。

分两部分讲解，一部分是通过继承自stream的实例来讲解。一部分是通过直接应用stream的使用来讲解。

Streams类型

有四种基本stream类型。

  1. Readable
    如：fs.createReadStream()
  2. Writable
    如：fs.createWriteStream()
  3. Duplex
    既是Readable又是Writable
    如：net.Socket
  4. Transform
    能够修改和改变的Duplex
    如：zlib.createDeflate()

Stream Implements API

在直接使用stream的时候，需要指明至少一个使用的基本类型。具体应用应指定继承的对象。

四种对应关系如下：

简单实现

简单实现无需使用继承。

实现writable stream

需要继承Writable

构造器

options有以下选项：

如：

writable._write(chunk, encoding, callback)

writable._writev(chunks, callback)

实现readable stream

需要继承Readable

new stream.Readable([options])

readable._read(size)

readable.push(chunk[, encoding])

实现duplex stream

需要继承Duplex

new stream.Duplex(options)

实现transform stream

需要继承Transform

new stream.Transform([options])

‘finish’ and ‘end’

transform._flush(callback)

transform._transform(chunk, encoding, callback)
