swift变量和方法

发表于 2017-01-18   |   分类于 Swift

定义

常量

常量和变量必须在使用前声明

let

let用来声明常量，不能改变。

变量

var

var用来声明变量。
声明变量和常量时，可指定类型：

print, println

打印到console面板上，println换行

字符串插值()

在字符串中使用变量和常量时，用 \()

注释

注释和js相同

Int

swift提供了8，16，32三种有符号和无符号整数类型：Int8/UInt8, Int16/UInt16, Int32/UInt32
一般直接使用Int就可以。

Double, Float

浮点数。
Double表示64位浮点数，高精度
Float表示32位浮点数。

类型转换

显式转换。即需要SomeType(value)

也有类似.toInt()之类的方法，类型转换有可能失败，所以需要进行if判断

类型别名 typealias

给现有类型定义一个别名

元组

  1. 多个值组合成一个组合值

  2. 赋值时可对应元组赋值

  3. 只需要部分元组值，用下划线填充：

  5. 还可以像json一样定义元组

可选赋值

  1. 可选变量。有一种情况为可选，即值可能缺失。在类型后加?

  2. 通过if或while语句来进行赋值，符合条件才赋值

只有可选才能使用if判断

  1. nil
    给可选变量赋值nil来清空值：

只能给可选变量使用
若声明变量时没有赋值，则自动被设置成nil

  1. 隐式可选
    在变量后加上!就是隐式可选。
    隐式可选是可选值，但是可以当做普通值来使用

断言assert

相当于做有条件的debugger

运算符

区间运算符…

即通过常用的数字，字母来表示数组值，常用于循环

  1. 闭区间运算符
    循环集合的所有值，如从a到b

  2. 半闭区间
    不包含集合的所有制，如从a到b，不包含b

字符串

特殊字符

  1. 转义\
    通过\转义的特殊字符：
    \0 空字符
    \ 反斜线
    \t 水平制表符
    \n 换行符
    \r 回车符
    \” 双引号
    \’ 单引号
  2. 单字节\x
    单字节Unicode标量，写成\xnn，其中nn为两位十六进制数
  3. 双字节\u
    双字节Unicode标量，写成\unnnn，其中nnnn为四位十六进制数
  4. 四字节\U
    四字节Unicode标量，写成\Unnnnnnnn，其中nnnnnnnn为八位十六进制数

空字符串

可通过 str.isEmpty() 来判断是否空字符串

计算长度

通过调用全局 countElements 函数来计算长度

uppercaseString/lowercaseString

字符串大小写

数组

数组普通格式为[value1, value2]
数组创建时可以指定元素类型

array.count

获取数组的长度

array.isEmpty

判断数组是否为空

array.append

数组末尾添加一个元素
也可直接相加，就可添加元素

这种相加也可直接相加一个数组：

修改数组元素

通过数组下标来修改。也可批量修改

不能通过下标设置超出数组长度的方式，来增加数组元素。

insert

用insert插入一个元素到atIndex指定的位置。

removeAtIndex

删除一个元素。并返回删除的元素

removeLast

删除数组最后的那个元素

数组for in

数组通过for in循环来迭代整个数组

enumerate

通过全局函数 enumerate 方法迭代数组，会得到索引和值

字典Dictionary

字典是储存类型相同元素的容器。每个元素是key value的键值对
其中key的要求是可哈希，即是唯一值。如String，Int，Double，Bool都是可哈希的值

添加元素

直接下标添加

更新元素 updateValue

直接下标更新。
或使用updateValue(forKey)来更新。

用updateValue()方法时，对应key存在时，就更新，不存在则添加元素

删除元素 removeValueForKey

返回被删除的值

  3. 迭代字典的value

while

循环开头计算条件

do while

if

switch

switch无需break。
若需要在执行一个case之后，接着进入下面的case，需要使用 fallthrough

switch where

在case中使用where来进一步判断

控制转移

使代码能跳转到别的部分: continue, break, fallthrough, return

函数

swift的函数，可指定参数的类型，和返回值的类型

返回值

当函数没有指定返回值时，实际上返回了一个空的元组。
也可返回多个值

参数

  1. 内部参数
    内部参数即在函数定义内部使用的，一般为Int，String等

  2. 外部参数
    外部参数主要作用为让参数的意义更加明显。
    需要在函数定义时，设定外部参数，然后调用的时候也要加上

外部参数快捷定义：#
当外部参数和参数名重合时，使用#

  1. 参数默认值
    在函数定义时，可给参数一个默认值

  2. 可变参数
    参数个数不定时，可加上…
    此时，常用循环来解析参数

  3. 变量参数
    函数的参数一般都是常量，若想让参数变为变量，可给参数添加var

  4. 输入输出参数 inout
    改变参数，并返回参数。
    函数定义处在参数前添加 inout，调用处加&

  5. 函数类型参数
    参数为一个函数，传参时要传入符合格式的函数，无需带参数

函数中可嵌套函数
